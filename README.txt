폴란드 교양강의 — 첫 여행자를 위한 6회 시리즈 (완성)
=====================================================

이 폴더는 자체 완결된 강의 핸드아웃 패키지입니다.
인터넷 연결 없이 (오프라인) 그대로 열어볼 수 있습니다.


여는 방법
---------
1. 이 폴더를 통째로 압축 해제합니다.
2. `index.html` 파일을 더블클릭하거나, 브라우저로 끌어다 놓습니다.
3. 메인 페이지에서 1–6회 중 한 강의를 선택합니다.


6강 시리즈 구성 (각 60분)
-------------------------
1회 · 폴란드라는 나라 — 첫인상과 좌표
        한국에서 "동유럽", 폴란드인은 "중부유럽". 좌표·사람·상징.
        + 인터랙티브 슬라이드 시안 (1회차만 별도 보관)

2회 · 천 년의 역사 — 사라졌던 나라가 어떻게 돌아왔는가
        966 세례 → 1410 그룬발트 → 1569 연방 → 1683 빈 →
        1772·1793·1795 분할 → 1918 부활 → 1939 침공 →
        1944 봉기 → 1989 자유선거 → 1999 NATO · 2004 EU

3회 · 쇼팽의 나라, 영화의 나라 — 문화와 예술
        쇼팽 · 미츠키에비치 · 노벨문학상 5인 ·
        바이다와 키에슬로프스키 · 코페르니쿠스 · 마리 퀴리

4회 · 도시를 걷다 — 4대 도시 + 일상·음식·매너
        크라쿠프 · 그단스크 · 브로츠와프 · 자코파네 +
        피에로기 · 즈워티 · 폴란드어 한 마디

5회 · 바르샤바 집중 — 도시의 역사·구역·이야기
        1596 수도 이전 → 1944 도시 85% 파괴 →
        1950s 카날레토 그림으로의 재건 → 오늘의 5구역

6회 · 바르샤바 가이드 투어 — 도보 + 트램 1회 루트
        중앙역 → 문화과학궁전 → 사스키 광장 →
        코페르니쿠스 → 바르샤바 대학 → 쇼팽의 심장 교회 →
        노비 시비아트 → 왕궁 광장 → 왕궁 → 인어상 →
        바르비칸 → 게토 추모비
        12 정거장, 6km, 4–5시간. 봉기 박물관 추천 연장.

기술 사양
---------
- 외부 의존성 0 (CDN 사용 안 함, 인터넷 없어도 동작)
- 시스템 폰트 (Apple SD Gothic Neo / Pretendard / Noto Sans KR)
- 모든 이미지 동봉 (총 약 65개)
- ES2018+ 표준 JS, 별도 빌드 없음
- 반응형 — 모바일·태블릿 폭에서 자동 1단 레이아웃

시안 두 가지에 대해
-----------------
1회차에 한해 두 가지 시안이 함께 들어 있습니다:

- 시안 A · 읽기 핸드아웃 (모든 회차)
  교양서 스타일의 정적 페이지. 좌측 sticky 미니목차,
  Beat 단위 설계(리드인→이미지→본문→정리).

- 시안 B · 인터랙티브 슬라이드 (1회차만)
  한 뷰포트 고정, → 키 / Space로 본인 템포 진행.
  23 슬라이드. 2회차 이후는 시안 A 단일 트랙으로 통일.


실제 여행 시 유의
---------------
- 6회 가이드 투어의 입장료·운영시간·트램 노선·환율은
  시기별 변동입니다. 출발 직전 공식 사이트 확인 권장.
- 박물관 일부는 월요일·화요일 휴무가 있으니 일정 확인.
- 6km 도보는 편한 신발 + 물 챙기기.
